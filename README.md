# 📌 Smart Leave Management System

A modern and industry-ready **Smart Leave Management System** built using **React.js, Redux Toolkit, and Redux Thunk**. This application allows users to apply for leave, manage requests, and track leave records efficiently with real-time UI updates.

---

## 🚀 Features

### ✅ Apply Leave
Users can apply for different types of leave:
- Sick Leave
- Casual Leave
- Emergency Leave
- Vacation Leave
- Half-Day Leave

Each request includes:
- Employee Name
- Leave Type
- Start Date
- End Date
- Reason
- Status

---

### 📊 View Leave Records
- Display leave requests in table/card layout
- Shows:
  - Employee Name
  - Leave Type
  - Duration
  - Status
  - Applied Date

---

### ✏️ Update Leave
- Modify leave type
- Update dates
- Edit reason

---

### ❌ Delete Leave
- Remove unwanted or cancelled leave requests
- Instant UI update using Redux

---

### 🔄 Leave Status Management
- Pending
- Approved
- Rejected
- Cancelled

---

### 🔍 Search & Filter
Filter leave records by:
- Employee Name
- Leave Type
- Status
- Date

---

### ⚡ Real-Time UI Sync
- No manual refresh required
- Redux handles instant UI updates

---

## 📚 Use Case

This system can be used in:
- Companies
- Schools
- Institutes

### Example Workflow:
1. Employee applies for leave
2. Manager reviews the request
3. Admin tracks leave history

---

## 🛠 Tech Stack

- **Frontend:** React.js
- **State Management:** Redux, Redux Toolkit
- **Middleware:** Redux Thunk
- **Styling:** Tailwind CSS / Material UI
- **Deployment:** Vercel / Netlify

---

## 🔥 Redux Modules

- applyLeave
- fetchLeaves
- updateLeave
- deleteLeave
- approveLeave
- loadingState
- errorHandling
